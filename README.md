GORGE
=====

Endless shoot 'em up game.

### Controls ###
<table>
	<tr><td>arrow keys</td>			<td>move</td></tr>
	<tr>
		<td><code>x</code></td>
		<td>shoot</td>
	</tr>
	<tr>
		<td><code>y</code> or <code>z</code></td>
		<td>activate force field/blast</td>
	</tr>
</table>
